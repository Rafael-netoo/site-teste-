Análise da Produção Científica da UFRPE

Integração de dados CAPES + Scopus utilizando RDF e SPARQL

Sobre o Projeto

Este projeto analisa a relação entre o tempo de carreira dos docentes da UFRPE e o impacto de sua produção científica.


Os dados foram integrados a partir das bases CAPES e Scopus, convertidos para RDF (.ttl) e consultados utilizando SPARQL.

Principais Insights

Observou-se tendência de crescimento no número de citações conforme aumenta o tempo de carreira dos docentes.
Alguns docentes com menos tempo de carreira apresentaram elevado impacto médio por publicação.
Maior quantidade de publicações não implica necessariamente maior impacto científico.
O H-index mostrou-se um indicador equilibrado para medir consistência científica.
