Análise da Produção Científica da UFRPE

Integração de dados CAPES + Scopus utilizando RDF e SPARQL

Sobre o Projeto

Este projeto tem como objetivo analisar a relação entre o tempo de carreira dos docentes da UFRPE e o impacto de sua produção científica.

Os dados foram integrados a partir das bases CAPES e Scopus, convertidos para RDF (.ttl) e consultados utilizando SPARQL.

Principais Insights

Observou-se uma tendência de aumento no número de citações conforme cresce o tempo de carreira dos docentes.
Docentes com menor tempo de carreira também apresentaram alto impacto médio por publicação.
Maior quantidade de publicações não implica necessariamente maior impacto científico.
O H-index mostrou-se um indicador mais equilibrado para medir consistência científica.
